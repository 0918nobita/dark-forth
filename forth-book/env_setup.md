## 環境構築

実行環境として Gforth を導入します。

### macOS を利用している方

Homebrew を利用してインストールできます。

```bash
brew install gforth
```

### コラム1: Forth 処理系探訪

Forth の処理系は多数存在しており、実行形式や対応している標準仕様がそれぞれ異なっています。
